# Quick Docker Desktop Installation

## Download & Install

1. **Download:** https://www.docker.com/products/docker-desktop/
2. **Run installer** - Accept defaults
3. **Restart computer**
4. **Wait for Docker to start** (whale icon in system tray)

## Verify Installation

```cmd
docker --version
docker-compose --version
```
